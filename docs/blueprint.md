# **App Name**: Bare Act Reader

## Core Features:

- Act List: Display a list of major Indian legal acts, tappable to navigate to chapters.
- Hierarchical Navigation: Hierarchical drill-down navigation to chapters and sections with accordion-style display.
- Section Detail View: Dedicated section view with breadcrumbs, formatted text, annotations, and sequential navigation.
- Offline Functionality: Offline-first functionality with bundled legal act data.
- Search Functionality: Robust search feature to query act titles, chapter titles, section numbers, and full text.
- Section Actions: Essential section actions: copy, share, bookmark/favorite, and adjust font size.

## Style Guidelines:

- Primary color: A calm blue (#4A8FE7), evoking trust and legal authority.
- Background color: A very light blue (#F0F8FF), provides a clean and calming backdrop for reading.
- Accent color: A subdued green (#77DD77), will complement the primary while setting apart key functionality and interactive elements, such as copy and share buttons.
- Body and headline font: 'Literata' serif, for readability and a traditional, reliable feel.
- Simple, monochrome icons for actions (copy, share, bookmark) for clarity.
- Minimalist layout with a focus on readability; ample whitespace.
- Subtle transitions for opening/closing chapters and sections; a smooth experience.
